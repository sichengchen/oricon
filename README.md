# Oricon Charts (1968 - 2008)

Oricon Weekly Top Singles (1968 - 2008)

[Live Demo](https://oricon.scchan.moe)

# 概要
- オリコン週間シングルランキング（1968-2008）
- 週ごとのランキング再生
- 速度切替
- スライダーで週をスクラブ

## Getting Started
```bash
npm install
npm run dev
```

Build / Preview:
```bash
npm run build
npm run preview
```
